# Example assets

Original project assets:

- `synthetic-scientific-figure.png` and the `gallery/` figures are generated locally by the included Python scripts.
- `palette-preview.png` is generated by the project's extractor from the synthetic figure.
- `hero-palette-intelligence.png` is original project artwork created for this repository; it contains no publisher material, journal figures, logos, or trademarks.

Licensed publication assets:

- [`licensed-figures/`](licensed-figures/) contains four official journal-figure renditions and their derived palette outputs.
- Each case records its exact CC BY 4.0 evidence, DOI, source URL, retrieval date, SHA-256, attribution, and changes in `metadata.yml`.
- These files are deliberately excluded from the repository's MIT licence. See [`../THIRD_PARTY_NOTICES.md`](../THIRD_PARTY_NOTICES.md).

Any future third-party example must stay in a separate case directory with equivalent licence, attribution, DOI, source URL, and modification records.
